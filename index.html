Architecture

  
Three-stage pipeline

  
Classic interpreter design, built from scratch in a single Python file with no external dependencies.


  

    

      
1

      
🔬

      
Lexer

      
Single-pass scanner. Characters → token stream. Handles strings, template literals, hex, octal, binary.

    

    
→

    

      
2

      
🌳

      
Parser

      
Recursive descent with full operator precedence. Produces lightweight AST nodes as Python dicts.

    

    
→

    

      
3

      
⚡

      
Evaluator

      
Tree-walking interpreter with lexical scoping, closures, prototype chain, and JS-spec type coercion.

    

    
→

    

      
4

      
🖥️

      
Output

      
console.log/error/warn routed to stdout. Errors include type and description.

    

  


  
What's supported

  

    
📦
Variables & Scope
let, const, var with hoisting. Lexical scoping via environment chain. Block scope enforced.

    
🔁
Control Flow
if/else, switch/case, for, while, do…while, for…of, for…in, break, continue, labels.

    
🎯
Functions
Declarations, expressions, arrow functions, closures, rest params, spread args, defaults.

    
🏛️
Classes
class keyword, constructor, methods, new, this binding, extends, prototype chain.

    
🧩
Destructuring
Array and object destructuring, rest patterns, spread in calls and array/object literals.

    
🛡️
Error Handling
try/catch/finally, throw, TypeError, RangeError, SyntaxError, ReferenceError.

    
📚
Built-ins
Full Array & String methods, Math, Date, JSON, parseInt, parseFloat, typeof, instanceof.

    
🔢
Number Literals
Hex 0xFF, octal 0o17, binary 0b1010, scientific notation, NaN, Infinity, −0.
